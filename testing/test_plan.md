# План тестирования React Store App

## Области тестирования:
1. Модульное тестирование компонентов
2. Интеграционное тестирование роутинга
3. Функциональное тестирование сценариев

- 95% покрытие основных компонентов

- Положительные отзывы от 80% тестировщиков
